#  AeroNova Prime

AeroNova Prime is an advanced Discord music bot developed by **Sanerex Development**, designed to deliver a premium music experience with powerful analytics, customization, and community-focused features.

##  Features

* No Prefix Access
* Custom Bot Profile Picture
* Custom Bot Banner
* Music XP System
* Music Ranks & Levels
* Global & Server Leaderboards
* Real-Time Listening Timeline
* Listening History Tracking
* Advanced User Profiles
* Song Statistics & Analytics
* Top Listener Rankings
* Playlist Management System
* Advanced Audio Filters
* Premium Server Features
* 24/7 Music Mode
* DJ Role System
* Vote Skip System
* Queue Management
* Lyrics Support

##  Statistics

AeroNova Prime provides detailed music analytics including:

* Total Songs Played
* Listening Time Tracking
* Global Song Charts
* User XP Progression
* Top Songs
* Top Listeners
* Music Activity Insights
